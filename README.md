The repository contains files of a project of a router to outsource the computations 
of QOS, the determination of network parameters as well as the convenient algorithm to be used 
by the router. The router gets telemetry parameters and sends them to the server, which then
computes and sends back the correct parameters to the router. It therfore applies those according the 
current environnement.
This is written in python for FASTAPI.
